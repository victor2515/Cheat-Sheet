CSS: Selectors

Selectors

A selector is used to target the specific HTML element(s) to be styled by the declaration.

The type selector selects all elements of a given type.


Universal
Explanation Example
The universal selector uses the * character to select all elementsof any type * {
    font-family: Verdana;
}
